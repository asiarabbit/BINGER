Detailed Description

store PID result using 3D tracks, used to facilitate parameter passing

The global event processor class, a singleton class, responsible for distribution of channel data to the detector hierarchy and the right channel objects. After the event is fully assigned, event assembly routine would be initiated. It is also this class' duty to provide analysis result output interface.
